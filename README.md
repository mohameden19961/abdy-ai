## 🚀 Démarrage rapide

```bash
# 1. Cloner le projet
git clone https://github.com/mohameden19961/abdy-ai.git
cd abdy-ai

# 2. Installer les dépendances
npm install

# 3. Lancer en local
npm start
```

> 🔑 Obtiens ta clé API gratuite sur [openrouter.ai/keys](https://openrouter.ai/keys) et configure-la dans ⚙️
